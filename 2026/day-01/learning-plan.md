## **90‑Day Goals**

### **Deploy a Production‑Grade Application**

- Containerize an application using Docker  
- Deploy on Kubernetes (local + cloud)  
- Implement CI/CD pipeline  
- Use Infrastructure as Code (Terraform)  
- Host on AWS or Azure  

---

### **Build End‑to‑End CI/CD Automation**

- Create pipelines using GitHub Actions / Jenkins  
- Automate build, test, scan, and deploy  
- Integrate Docker + Kubernetes deployments  
- Implement environment-based deployments  

---

### **Cloud Infrastructure Deployment**

- Provision cloud infrastructure using Terraform  
- Deploy app to AWS (EC2/EKS) or Azure (VM/AKS)  
- Configure monitoring & logging  
- Implement basic security best practices  

---

## **Core DevOps Skills to Build**

### **Linux & Shell Scripting**

- Linux administration fundamentals  
- Process, networking, permissions  
- Write automation scripts  

---

### **Containers & Orchestration**

- Docker (images, volumes, networking)  
- Kubernetes (Pods, Deployments, Services, Ingress, Helm basics)  

---

### **CI/CD & Cloud Infrastructure**

- GitHub Actions / Jenkins  
- Terraform (IaC)  
- AWS or Azure fundamentals  

---

## **Weekly Time Budget**

- **Weekdays:** 3–3.5 hours per day (Mon–Fri)  
- **Weekends:** 4–6 hours per day  
- **Total Weekly Commitment:** ~22–25 hours
